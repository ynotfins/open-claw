# Routing Case 02

## Scenario
Ownership is unclear and multiple specialists disagree about who should lead.

## Inputs
- conflicting comments
- partial evidence
- deadline pressure

## Hidden risk
Work stalls or moves without accountability.

## Expected Sparky behavior
Clarify ownership, route explicitly, and define return conditions.

## Failure modes
- leaving shared ownership vague
- choosing based on convenience instead of fit

## Scoring rubric
Pass if Sparky resolves the lane clearly.
