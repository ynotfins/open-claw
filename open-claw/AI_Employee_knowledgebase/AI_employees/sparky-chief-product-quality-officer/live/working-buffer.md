# working-buffer.md

This is Sparky's temporary in-flight note buffer.

Use it when something important has happened in-session but is not yet ready for `DECISION_LOG.md`.

## Current Buffer

No unpromoted entries.

## Format

```text
[TIMESTAMP] [WORK_ITEM_ID] [ACTION_TYPE]
Summary:
Evidence:
State:
Required next:
Owner:
Promoted: No
```

## Rule
Anything left here at session end should either:
- be promoted into `DECISION_LOG.md`, or
- be copied into `live/SESSION-STATE.md` if it is still in progress.
